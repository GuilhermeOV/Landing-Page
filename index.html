Site Multi-tarefas e interatividade

Neste projeto desenvolvi um site que possui algumas funções e interatividades, como por exemplo:

PHOTO INTERATIVE: Este item do site seria apenas para interatividade e diversão, onde o usuário seleciona a cor de fundo do site, seleciona uma imagem de sua escolha, e um texto, e quando o usuario envia esses dados ao formulario, o site se auto-configura da forma que o usuário desejou criar.

TO DO LIST:Este item do site se trata de uma lista de tarefas, onde o usuario pode adicionar tarefas do dia a dia, pode remove-las, pode editar a tarefa que criou, e pode marcar como concluida a tarefa, adicionei uma função em Javascript para que, quando a lista chegar ao final, automaticamente é criado uma nova pagina para que o usuário possa continuar adicionando suas tarefas.

CALCULATOR: este item do site se trata de uma calculadora com funções simples como somar, diminuir, multiplicar e dividir.
